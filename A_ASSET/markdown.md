# 마크다운이란 무엇인가?
`#`은 .md에서 제목을 표기한다.
## 다른 제목을 작성한다.
### 다시 다른 제목을 작성한다.

`#`의 갯수는 제목의 레벨을 의미한다. 갯수가 많을수록 중요도가 낮아짐.

---
내용을 표시 할 때는 어떤 형태가 있는가? <br/>또한, 어떤 표시를 하는가?

> 인용문: 문장 또는 단락을 강조처리 한다.
---
문장을 작성 할 때 유용한 마크다운: <br/> **굵은 글씨** *기울임* ~~취소선~~ <ins>밑줄표시</ins> <br/>
백팁을 사용하면 코드 노출이 가능하다
`<ins>밑줄사용<ins>`
